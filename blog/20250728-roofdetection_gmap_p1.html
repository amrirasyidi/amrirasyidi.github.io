Fetching the data

Google provides an API specifically to fetch imagery, it is covered under the "Maps Static API", which you can enable by following the guide in this page. If you're just trying things out, you shouldn't worry too much about the pricing, as long as your "trying" don't exceed 10,000 API calls (if I correctly intepreted this page).

To programmatically fetch the satellite imagery using Python, we can use request. On the documentation, the request url would look like this: 

            https://maps.googleapis.com/maps/api/staticmap?center=40.714728,-73.998672&zoom=12&size=400x400&maptype=satellite&key=YOUR_API_KEY&signature=YOUR_SIGNATURE

To use it in request, we could use something like this 

base_url = "https://maps.googleapis.com/maps/api/staticmap"

# Build API request parameters
params = {
    'center': f"{latitude},{longitude}",
    'zoom': Config.ZOOM_LEVEL,
    'size': Config.IMAGE_SIZE,
    'maptype': Config.MAP_TYPE,
    'format': Config.IMAGE_FORMAT,
    'key': Config.api_key
}

# Make API request
response = requests.get(base_url, params=params, timeout=30)

In the case above, Config is a python class which contains the relevant parameters to be fed in the request.
