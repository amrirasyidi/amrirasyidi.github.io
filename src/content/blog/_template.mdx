## Text Formatting

**bold text**, *italic text*, ~~strikethrough~~, `inline code`

Markdown handles 90% of text. For special styling, use raw HTML:

<p style="color: #f97583;">Colored text with inline style.</p>
<p style="text-align: center;"><strong>Centered text</strong></p>

### Tooltip

Hover for extra info: <Tooltip term="technical term">Explanation appears on hover. Supports **bold**, *italic*, `code`, and <br />line breaks inside.</Tooltip>

## Headings

## H2 — Main Sections
### H3 — Subsections
#### H4 — Sub-subsections

All headings auto-generate IDs via rehype-slug (e.g. `#main-sections`).
Link to them: [jump to tables](#tables).

## Lists

Numbered:

1. First step
2. Second step
3. Third step with nested:
   1. Sub-step A
   2. Sub-step B

Bullets:

- Key point one
- Key point two
- Key point with nested:
  - Detail A
  - Detail B

## Code

Inline: `variable_name`, `function()`, `pip install package`

Fenced code block with syntax highlighting (Shiki, build-time):

```python
import pandas as pd

df = pd.read_csv('data.csv')
print(df.head())
```

```sql
SELECT *
FROM table_name
WHERE condition = 'value';
```

```bash
pip install osmnx
conda install -c conda-forge geopandas
```

File tree (plain text block):

```
.
├── db
│   ├── datareading.db
│   └── datareading.db.tmp
├── notebook
│   └── analysis.ipynb
└── sql
    └── query.sql
```

## Images and Media

### Using Figure component (recommended)

<Figure src="/assets/blog/images/your-image.jpg" alt="Description">
  Caption describing the image content
</Figure>

### With ID for cross-referencing

<Figure id="important-diagram" src="/assets/blog/images/diagram.jpg" alt="Important diagram">
  This diagram can be [referenced later](#important-diagram)
</Figure>

### Raw HTML for custom styling

<img src="/assets/blog/images/your-image.jpg" alt="Description"
     style="border-radius: 8px; display: block; margin: 0 auto; width: 100%; max-width: 400px;" />

### Video

<video controls style="width: 100%; max-width: 600px; display: block; margin: 0 auto;">
  <source src="/assets/blog/videos/your-video.mp4" type="video/mp4" />
  Your browser doesn't support video.
</video>

## Tables

Simple markdown table:

| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |

Complex table (use raw HTML — supports bold, tooltips, multi-line):

<table style="width: 100%; border-collapse: collapse; margin: 20px 0;">
  <thead>
    <tr>
      <th>Feature</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Bold cell</strong></td>
      <td>
        Multi-line content with <Tooltip term="tooltip">Works inside tables too!</Tooltip>
      </td>
    </tr>
  </tbody>
</table>
